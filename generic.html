Destroy Voxel

What was implemented in this project:

1. Simple creation of a destructible block when creating a level.

2. Simple creation of a voxel block destroyer. Can be used on any item with customizable parameters.

3. Management is not great, since it is only needed to test destructibility and build a general picture and the possibility of developing an idea. A character with the implementation of movement, jump, and turn behind the camera.

4. A simple implementation of weapons with customizable parameters: rate of fire, the number of bullets fired per shot.

5. Lightwork has been done and post-processing added.

It is possible to adjust the zone of destructibility, the force of destruction, and the scattering of particles. The script is easy to use any object with this code can become a severe cube destroyer! :3

It is also easy to edit the size of the destructible object
